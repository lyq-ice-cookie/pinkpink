git commit -m "说明"     # 2. 提交到本地仓库（"说明"换成你这次改了什么）
git push origin main     # 3. 推到你 GitHub 的 main 分支

**git add . 是一次性打包当前目录所有变更，而 git add 具体文件名 则是精确挑选单个文件**。
